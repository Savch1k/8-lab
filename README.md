## Название приложения
**Snake Game** - классическая аркадная игра "Змейка", реализованная на Python с использованием библиотеки Tkinter.

## Задание
Реализуйте приложение с GUI по своему варианту. Можно изменить задание на собственную тему, согласовав с преподавателем. Требования:
   приложение должно быть написано с применением ОО парадигмы
   исключительные ситуации должны обрабатываться с использованием собственных исключений
### Демонстрация принципов ООП в коде:

| Принцип | Реализация |
|---------|------------|
| **Наследование** | Исключения `InvalidMoveError`, `GameOverError` наследуются от `SnakeGameError` |
| **Инкапсуляция** | Приватные атрибуты (`_body`, `_direction`), геттеры через `@property` |
| **Полиморфизм** | Перегрузка операторов `__add__`, `__eq__` в классе `Point` |
| **Композиция** | Класс `SnakeGame` содержит объекты `Snake` и `Food` |
| **Исключения** | Пользовательские типы исключений для различных ошибок |

### Управление

| Действие | Клавиши |
|----------|---------|
| Движение вверх | ↑ (Стрелка вверх) или W |
| Движение вниз | ↓ (Стрелка вниз) или S |
| Движение влево | ← (Стрелка влево) или A |
| Движение вправо | → (Стрелка вправо) или D |
| Пауза / Продолжить | P |
| Новая игра | R |

### Правила игры

1. Управляйте змейкой, чтобы собирать красные кружки (еду)
2. Каждая съеденная еда приносит **10 очков**
3. Змейка увеличивается в длину после каждой съеденной еды
4. Игра заканчивается, если змейка:
   - Врезается в стену игрового поля
   - Врезается в собственное тело
5. **Победа** наступает, когда змейка заполняет всё игровое поле
6. Скорость игры постепенно увеличивается с ростом счёта
