## 📱 Название приложения

**Змейка (Snake Game)** — классическая аркадная игра, реализованная на Python с использованием объектно-ориентированного подхода и графического интерфейса Tkinter.

---

## 📖 Описание

"Змейка" — это известная аркадная игра, в которой игрок управляет змейкой, движущейся по игровому полю. Цель игры — собирать еду, увеличивая длину змейки и набирая очки, избегая столкновения со стенами и собственным телом.

### Особенности программы:

- 🎮 **Классический геймплей** — управляйте змейкой, собирайте еду и становитесь длиннее
- 🎨 **Красочный интерфейс** — анимированная змейка с глазами, разноцветное поле
- ⚡ **Динамическая сложность** — скорость игры увеличивается с каждыми 10 очками
- 🏆 **Система победы** — при заполнении всей клетки змейкой вы побеждаете
- ⏸️ **Пауза** — возможность приостановить игру в любой момент
- 🔄 **Перезапуск** — быстрый старт новой игры без закрытия окна
- 🖱️ **Управление с клавиатуры** — стрелки или WASD

### Технические особенности:

- ✅ Полное соблюдение **объектно-ориентированной парадигмы**
- ✅ Использование **собственных исключений** для обработки ошибок
- ✅ Разделение логики и представления (архитектура MVC)
- ✅ Инкапсуляция, наследование, полиморфизм и композиция
- ✅ Аннотации типов для лучшей читаемости кода

## 🚀 Инструкция по запуску

### Системные требования

- **Python** версии 3.6 или выше
- Библиотека **Tkinter** (входит в стандартную установку Python)

### Пошаговая инструкция

#### 1. Убедитесь, что Python установлен

Откройте терминал (командную строку) и выполните:

```bash
python --version
```

или

```bash
python3 --version
```

Если Python не установлен, скачайте его с [официального сайта](https://www.python.org/downloads/)

#### 2. Сохраните код программы

Скопируйте полный код игры (предоставленный выше) в файл с именем `snake_game.py`

#### 3. Запустите игру

В терминале перейдите в папку с файлом и выполните:

```bash
python snake_game.py
```

или

```bash
python3 snake_game.py
```

#### 4. Наслаждайтесь игрой! 🎮

| <kbd>R</kbd> | Новая игра (после окончания) |

### Правила игры

1. **Цель** — набрать как можно больше очков, собирая красные ягоды 🍎
2. **Рост** — каждая съеденная ягода увеличивает длину змейки на 1 сегмент
3. **Очки** — за каждую ягоду начисляется 10 очков
4. **Скорость** — каждые 10 очков игра ускоряется (максимальная скорость на 80 мс)
5. **Поражение** — игра заканчивается, если змейка:
   - Врезается в стену
   - Врезается в собственное тело
6. **Победа** — игра заканчивается победой, если змейка заполнила всё игровое поле

- **Зелёная/бирюзовая змейка** — главный персонаж (голова светится)
- **Красная ягода** — еда, которую нужно собирать
- **Счёт** — текущее количество очков в правом верхнем углу
- **Кнопка "Новая игра"** — перезапуск игры мышью
- **Статусная строка** — подсказки и статус паузы

### Пример игрового процесса

1. Змейка начинает движение вправо
2. Управляйте стрелками, чтобы подвести голову к ягоде
3. При съедании ягоды змейка растёт, появляется новая ягода
4. Старайтесь не врезаться в стены и своё тело
5. Чем больше очков — тем быстрее игра
6. При поражении или победе нажмите <kbd>R</kbd> для новой игры

## 🛠️ Возможные проблемы и решения

### Проблема: "tkinter не найден"

**Решение:** Tkinter обычно идёт в комплекте с Python. Если его нет, установите:

- **Ubuntu/Debian:** `sudo apt-get install python3-tk`
- **Fedora:** `sudo dnf install python3-tkinter`
- **Windows/Mac:** Переустановите Python, убедившись, что Tkinter выбран в компонентах

### Проблема: Игра слишком быстрая/медленная

**Решение:** Скорость регулируется автоматически. Начальная скорость — 150 мс между кадрами, максимальная скорость (при 350+ очках) — 80 мс.

### Проблема: Не работают клавиши управления

**Решение:** Убедитесь, что окно игры активно (на него кликнули). Попробуйте использовать альтернативное управление (WASD).
